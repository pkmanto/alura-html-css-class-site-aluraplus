# Projetos de aula em HTML/CSS do curso da Alura

Este projeto foi criado à partir das aulas de HTML/CSS da Alura. A ideia é montar um site com formato de landing, utilizando as estruturas de Section e DIV no HTML, além de utilizar as 'margins' e 'flexbox' do CSS. A combinação final transformou em um site réplica do Alura+.
